🛠 הגדרות Firebase

הוראות הגדרה ראשונית (חינם לחלוטין):
  1. כנס ל-console.firebase.google.com
  2. Add project → שם → אשר
  3. Build → Realtime Database → Create Database → אירופה → test mode
  4. ⚙ Project Settings → General → Your apps → </> → Register app
  5. העתק apiKey, databaseURL, projectId למטה
  6. בחר מפתח סוכנות ייחודי (למשל: shirel2024)
